 php多进程常驻shell
 ====
 * 多进程shell优点：
 * --------------------------------------------------------------
 * 使用多进程,子进程结束以后,内核会负责回收资源                          
 * 使用多进程,子进程异常退出不会导致整个进程Thread退出,父进程还有机会重建流程 
 * 一个常驻主进程,只负责任务分发,逻辑更清楚                             
 * --------------------------------------------------------------
 *
 * 单进程shell弊端：
 * ----------------------
 * 没有专门的gc例程        
 * 也没有有效的内存管理途径 |
 * ----------------------
 *
 * 参考文献：http://www.laruence.com/2009/06/11/930.html
